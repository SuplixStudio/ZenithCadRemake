Keybinds

  • Tab - Edit mode
  • W/E/R - Move/Rotate/Scale
  • G - Extrude (edit mode)
  • S - Subdivide
  • F - Toggle face/vert select
  • Ctrl+Z/Y - Undo/Redo
  • Ctrl+E - Script editor
  • K - Animation editor
  • Ctrl+S - Save project
  • Ctrl+I - Import model
  • Middle mouse - Orbit
  • Right mouse - Pan
