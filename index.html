Documentation

FabsdRecording is designed to be intuitive. Here is a quick guide on how to use the advanced features.

How to Record a Specific Window

Instead of recording your entire screen, you can target a specific application window.

  1. Launch FabsdRecording.
  2. In the "Source" dropdown, select Window Capture.
  3. Click the Select Window button that appears.
  4. A list of open applications will appear. Select the one you want to record (e.g., "Notepad" or "Chrome").
  5. Click OK. The preview will update to show only that window.

Preventing the "Infinite Tunnel"

If you are recording the full screen that FabsdRecording is running on:

  • Check the box labeled Hide Window from Capture.
  • The FabsdRecording window will disappear from the recording, revealing whatever is behind it.
  • It remains visible to you, so you can still control the recording.

FAQ & Troubleshooting

Q: The window list is empty?
A: Make sure the application you want to record is not minimized to the tray. It must be open on the desktop.

Q: Audio is out of sync?
A: We are working on this for version 1.1. For now, try to keep recordings under 2 hours.
