FormnexoStudio Documentation

Complete guide to 3D modeling for game assets

Getting Started

Installation

Download and run form.exe. No installation required - it's portable! The application will create a config folder in your user directory on first run.

First Launch

When you first open FormnexoStudio, you'll see:

  • Scene Hierarchy - Left panel showing all objects
  • Properties Panel - Bottom left with object details
  • 3D Viewport - Center area for modeling
  • Toolbar - Top bar with tools and modes
  • Status Bar - Bottom showing camera position and stats

Saving Your Work

FormnexoStudio uses .form project files to save your entire scene:

  • Press Ctrl + S to quick save
  • Click Save in the menu bar to save as a new file
  • Click Load to open an existing project
  • Projects save all objects, materials, transforms, and scene hierarchy
Note: Texture files are not embedded in .form projects. Keep texture files in the same location when loading projects.

Interface Overview

Menu Bar

Top menu provides quick access to:

  • Save - Save current project (Ctrl+S)
  • Load - Open existing project
  • Import - Load FBX, OBJ, glTF, GLB files
  • Export - Save models to various formats
  • Add Primitives - Cube, Sphere, Plane, Cylinder
  • Undo/Redo - History management
  • Script Editor - Open Lua scripting panel

Toolbar

Main toolbar shows current mode and tools:

  • Move/Rotate/Scale - Transform gizmos
  • Solid/PBR Toggle - Switch rendering modes
  • Edit Mode Tools - Extrude, Subdivide, Inset, Weld

Properties Panel

Shows detailed information about selected object:

  • Transform values (Position, Rotation, Scale)
  • Mesh statistics (Vertices, Faces)
  • Material properties
  • Soft selection settings
  • Skeleton and animation data

Modeling Tools

Object Mode

Default mode for manipulating entire objects. Use gizmos to transform:

  • W - Move tool (translate)
  • E - Rotate tool
  • R - Scale tool
  • Delete - Delete selected object

Edit Mode

Press Tab to enter Edit Mode and modify mesh geometry.

Selection Modes

  • Vertex Mode - Select and move individual points
  • Edge Mode - Select edges (Alt+Click for edge loops)
  • Face Mode - Select polygons
  • Toggle with F key

Modeling Operations

Tool Shortcut Description
Extrude G Pull selected geometry outward. Move mouse to adjust, click to confirm, Esc to cancel
Subdivide S Split selected faces into smaller faces for more detail
Inset I Create an inset face within selected faces
Bevel B Round off edges and corners
Target Weld V Merge selected vertices together
Merge Vertices M Combine nearby vertices
Bridge Edge Loops J Connect two edge loops with faces
Subdivide Edges D Add vertices along selected edges
Collapse Edges X Remove edges and merge vertices

Soft Selection

Enable in the Properties panel to create smooth, organic deformations. Adjusts nearby vertices with falloff based on distance.

Materials & Textures

Material Editor

Click "Open Material Editor" in the Properties panel to access the fullscreen material editor.

PBR Properties

  • Base Color - RGB sliders for main color
  • Metallic - 0 = dielectric, 1 = metal
  • Roughness - 0 = mirror, 1 = matte
  • Alpha - Transparency (0 = invisible, 1 = opaque)
  • Emissive - Self-illumination color and strength

Material Presets

Quick-apply common materials:

  • Metal - Shiny metallic surface
  • Glass - Transparent with low roughness
  • Plastic - Smooth colored surface
  • Wood - Rough brown material
  • Rubber - Dark matte surface
  • Emissive - Glowing orange light

Texture Loading

In the material editor, use the TEXTURES section to load image files:

  • Albedo - Base color texture (diffuse map)
  • Normal - Surface detail without geometry
  • Metallic/Roughness - PBR texture maps

Supported formats: PNG, JPG, JPEG, BMP, TGA

Note: Textures are automatically loaded from embedded data in GLB files. For FBX/OBJ, place texture files in the same folder as the model.

Real-Time Preview

The material editor shows a live 3D preview of your selected object with the material applied. The preview rotates automatically and updates as you adjust sliders.

Import & Export

Importing Models

Click Import in the menu bar to open the import dialog.

Supported Formats

  • FBX - Autodesk format with animations
  • OBJ - Universal format (geometry only)
  • glTF/GLB - Modern format with PBR materials

The importer automatically:

  • Loads embedded textures from GLB files
  • Imports materials and assigns them to meshes
  • Loads skeletal animations and bone data
  • Triangulates all geometry
  • Generates normals if missing

Exporting Models

Select an object and click Export in the menu bar.

Export Formats

  • OBJ - Universal format with MTL material file
  • FBX - Exports as OBJ (full FBX export coming soon)
  • glTF - Exports as OBJ (full glTF export coming soon)

OBJ export includes:

  • Vertex positions with world transform applied
  • Normals and UV coordinates
  • MTL file with PBR material properties

Rendering Modes

Solid Mode (Default)

Fast Phong shading for modeling work. Good performance, simple lighting.

PBR Mode

Press Z or click the "PBR" button to enable physically-based rendering.

PBR Features

  • Cook-Torrance BRDF for realistic materials
  • Proper metallic and roughness response
  • Fresnel reflections
  • Multiple light sources (key + fill)
  • HDR tonemapping
  • Gamma correction

Use PBR mode to preview how materials will look in game engines like Unity or Unreal.

Animation System

Animation Editor

Press K to open the animation timeline.

Features

  • Play/Pause/Stop controls
  • Timeline scrubbing
  • Keyframe creation
  • Bone selection and posing
  • Multiple animation clips per object

Skeletal Animation

Imported models with skeletons automatically show bone gizmos. Select bones to pose and animate characters.

Lua Scripting

Script Editor

Press Ctrl + E to open the Lua script editor.

API Reference

Creating Objects

-- Create a cube at position (x, y, z) with size
local cube = api:CreateCube(0, 0, 0, 1.0)

-- Create a sphere
local sphere = api:CreateSphere(5, 0, 0, 2.0)

-- Create a plane
local plane = api:CreatePlane(0, -1, 0, 10.0)

-- Create a cylinder
local cylinder = api:CreateCylinder(0, 5, 0, 1.0, 3.0)

Procedural Generation

-- Generate a grid of cubes
for x = 0, 9 do
    for z = 0, 9 do
        api:CreateCube(x * 2, 0, z * 2, 1.0)
    end
end

-- Create a spiral
for i = 0, 50 do
    local angle = i * 0.5
    local x = math.cos(angle) * i * 0.2
    local z = math.sin(angle) * i * 0.2
    api:CreateCube(x, i * 0.1, z, 0.5)
end

Mesh Manipulation

-- Access scene nodes
local nodes = api:GetAllNodes()
for i, node in ipairs(nodes) do
    print("Node: " .. node.Name)
end

-- Modify transforms
api:SetPosition(node, 0, 5, 0)
api:SetRotation(node, 0, 45, 0)
api:SetScale(node, 2, 2, 2)

Example Scripts

Procedural City

-- Generate a simple city layout
for x = 0, 10 do
    for z = 0, 10 do
        if math.random() > 0.3 then
            local height = math.random(2, 8)
            api:CreateCube(x * 3, height / 2, z * 3, 1.0, height, 1.0)
        end
    end
end

Tips & Tricks

Workflow Tips

Edge Loop Selection

Hold Alt and click an edge to select the entire edge loop. Great for adding detail to cylindrical objects.

Soft Selection

Enable in Properties panel for organic modeling. Adjust radius to control falloff distance.

Multi-Select

Hold Shift and click to add to selection. Works in both Object and Edit modes.

Box Select

Click and drag to create a selection box. All vertices/faces inside will be selected.

Material Preview

Use PBR mode (Z) to see realistic material appearance before exporting to game engines.

Quick Primitives

Use menu bar buttons to quickly add cubes, spheres, planes, and cylinders to your scene.

Performance Tips

  • Use Solid mode for heavy modeling work, switch to PBR for final preview
  • Keep poly count reasonable - aim for game-ready assets
  • Use subdivision sparingly - it multiplies face count quickly
  • Merge vertices after modeling to remove duplicates

Game Asset Workflow

  1. Model in Solid mode for speed
  2. Apply materials and adjust PBR values
  3. Load textures (albedo, normal, metallic/roughness)
  4. Preview in PBR mode to check appearance
  5. Export to OBJ/FBX for Unity/Unreal
Pro Tip: FormnexoStudio is designed for game asset creation. Keep models optimized with clean topology and reasonable poly counts for real-time rendering.

Common Issues

Textures Not Loading

For FBX/OBJ files, ensure texture files are in the same folder as the model. GLB files with embedded textures load automatically.

Model Appears Grey

The model may not have textures assigned. Open the material editor and load textures manually, or check if the import included materials.

Extrude Not Working

Make sure you're in Edit Mode (Tab) and have geometry selected. Press G to start extrude, move mouse to adjust, click to confirm.

Can't Select Vertices

Toggle selection mode with F to switch between Vertex, Edge, and Face modes.
