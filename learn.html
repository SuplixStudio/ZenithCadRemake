Getting Started

BEGINNER

Tutorial 1: Interface & Navigation

Time: 5 minutes | Goal: Learn to navigate the 3D viewport

1 Launch FormnexoStudio
Run form.exe and wait for the splash screen. You'll see a default scene with a grid.
2 Camera Controls
• Hold Middle Mouse and drag to orbit around the scene
• Hold Right Mouse and drag to pan the camera
• Use Scroll Wheel to zoom in and out
3 Add Your First Object
Click the Cube button in the top menu bar. A cube will appear at the origin (0, 0, 0).
4 Transform Tools
• Press W for Move tool (see colored arrows)
• Press E for Rotate tool (see rotation rings)
• Press R for Scale tool (see scale handles)
• Click and drag the colored handles to transform
Pro Tip: Press H to toggle the shortcuts overlay - it shows all available hotkeys!

Your First Model: Simple Table

BEGINNER

Tutorial 2: Build a Table from Primitives

Time: 10 minutes | Goal: Learn basic object manipulation

1 Create the Tabletop
• Click Cube in the menu
• Press R for Scale tool
• Drag the green handle (Y-axis) down to make it flat
• Drag the red (X) and blue (Z) handles out to make it wider
2 Create First Leg
• Add another Cube
• Press R and scale it tall and thin (like a cylinder)
• Press W and move it under one corner of the tabletop
3 Duplicate the Legs
• Select the leg in the Scene Hierarchy (left panel)
• Press Ctrl + D to duplicate
• Move the duplicate to another corner
• Repeat for all four corners
4 Apply Materials
• Select the tabletop
• In Properties panel, click Open Material Editor
• Choose the Wood preset
• Click Apply and close the editor
Success! You've created your first 3D model. Try pressing Z to see it in PBR render mode.

Low-Poly Game Asset: Sword

INTERMEDIATE

Tutorial 3: Model a Low-Poly Sword

Time: 20 minutes | Goal: Learn Edit Mode and modeling tools

1 Start with a Cube
• Add a Cube to the scene
• Press Tab to enter Edit Mode
• You'll see the cube's vertices highlighted
2 Shape the Blade
• Press F to switch to Face selection mode
• Click the top face to select it
• Press R and scale it smaller (this will be the blade tip)
• Press G to extrude upward - move mouse up and click to confirm
3 Create the Handle
• Select the bottom face
• Press G to extrude downward
• Press R to scale it thinner (handle should be narrower than blade)
• Extrude again for the pommel (bottom grip)
4 Add the Crossguard
• Press Tab to exit Edit Mode
• Add a new Cube
• Scale it flat and wide
• Position it between the blade and handle
5 Apply Materials
• Select the blade, apply Metal material
• Select the handle, apply Wood or Rubber
• Select the crossguard, apply Metal
Pro Tip: Press Z to toggle PBR mode and see your metallic materials shine!

Textured Wooden Crate

INTERMEDIATE

Tutorial 4: Create a Textured Crate

Time: 15 minutes | Goal: Learn texture mapping and PBR materials

1 Model the Crate
• Add a Cube
• Press Tab for Edit Mode
• Press S to subdivide all faces once (adds detail)
• Press I to inset the faces slightly (creates panel look)
2 Add Edge Details
• Press F to switch to Edge mode
• Hold Alt and click an edge to select the loop
• Press B to bevel the edges (rounds them off)
• This makes the crate look more realistic
3 Exit and Apply Base Material
• Press Tab to exit Edit Mode
• Open the Material Editor
• Choose Wood preset as a starting point
4 Load Textures
• In the TEXTURES section, click Load Albedo
• Browse to a wood texture image (PNG/JPG)
• Optionally load Normal and Roughness maps
• The texture will appear on your crate immediately
5 Adjust PBR Values
• Set Roughness to 0.7-0.8 (wood is not shiny)
• Set Metallic to 0.0 (wood is not metal)
• Adjust Base Color tint if needed
• Click Apply
Note: If you don't have textures, you can find free PBR textures at sites like Poly Haven or Texture Haven.

Character Modeling Basics

ADVANCED

Tutorial 5: Simple Low-Poly Character

Time: 30 minutes | Goal: Learn organic modeling with soft selection

1 Start with Body
• Add a Cube for the torso
• Scale it taller than wide
• Add another cube for the head, position above torso
• Scale the head smaller
2 Add Limbs
• Add cubes for arms (2) and legs (2)
• Scale them long and thin
• Position them at shoulders and hips
• Add small cubes for hands and feet
3 Smooth with Soft Selection
• Select the torso and enter Edit Mode
• In Properties panel, enable Soft Selection
• Set radius to 2.0
• Move vertices to create organic curves
• Nearby vertices will move smoothly with falloff
4 Add Details
• Use S to subdivide faces for more detail
• Use G to extrude facial features
• Use I to inset for clothing details
• Keep it low-poly for game performance
5 Apply Materials
• Skin: Base Color (peach/brown), Roughness 0.5
• Clothing: Choose colors, Roughness 0.7
• Hair: Dark color, Roughness 0.3
Pro Tip: For game characters, aim for 500-2000 triangles. Use the Properties panel to check your poly count!

Mastering PBR Materials

INTERMEDIATE

Tutorial 6: Understanding PBR Workflow

Time: 15 minutes | Goal: Learn physically-based rendering principles

What is PBR?

Physically-Based Rendering simulates how light interacts with real materials. It uses these key properties:

Base Color (Albedo)
The raw color of the material without lighting. For metals, this should be a metallic color (gold, silver, copper). For non-metals, use any color but avoid pure white or black.
Metallic
• 0.0 = Non-metal (wood, plastic, skin, fabric)
• 1.0 = Metal (iron, gold, chrome)
• No in-between values in reality (but you can blend for artistic effect)
Roughness
• 0.0 = Perfect mirror (polished chrome)
• 0.3 = Glossy (car paint, polished wood)
• 0.7 = Matte (concrete, fabric)
• 1.0 = Completely rough (chalk, rough stone)
Emissive
Makes materials glow. Use for:
• Neon signs
• LED lights
• Glowing sci-fi elements
• Magic effects

Common Material Recipes

Polished Metal
Base Color: Metallic color (e.g., RGB 192, 192, 192 for steel)
Metallic: 1.0
Roughness: 0.2
Painted Plastic
Base Color: Any bright color
Metallic: 0.0
Roughness: 0.4
Rough Stone
Base Color: Grey/brown
Metallic: 0.0
Roughness: 0.9
Important: Press Z to toggle PBR mode and see your materials render correctly. Solid mode won't show metallic/roughness properly!

Procedural Generation with Lua

ADVANCED

Tutorial 7: Generate a Forest with Code

Time: 20 minutes | Goal: Learn Lua scripting for procedural content

1 Open Script Editor
Press Ctrl + E to open the Lua script editor panel.
2 Create a Tree Function
Type this code: 
-- Function to create a simple tree
function createTree(x, z)
    -- Trunk
    local trunk = api:CreateCylinder(x, 1, z, 0.3, 2)
    
    -- Leaves (sphere on top)
    local leaves = api:CreateSphere(x, 3, z, 1.5)
end
3 Generate a Forest
Add this code below: 
-- Create a 10x10 forest with random spacing
for i = 1, 50 do
    local x = math.random(-20, 20)
    local z = math.random(-20, 20)
    createTree(x, z)
end
4 Run the Script
Click the Execute button. You'll see 50 trees appear randomly in your scene!
5 Experiment
Try modifying:
• Tree size: Change cylinder and sphere dimensions
• Forest density: Change the loop count
• Spacing: Adjust the random range
• Add variation: Randomize tree heights
Challenge: Add random rotation to each tree so they don't all face the same direction!

Export to Game Engines

BEGINNER

Tutorial 8: Export to Unity/Unreal

Time: 10 minutes | Goal: Get your models into a game engine

1 Prepare Your Model
• Make sure materials are applied
• Check poly count in Properties panel (keep it reasonable)
• Position model at origin (0, 0, 0) for easy import
• Apply any final transforms
2 Export the Model
• Select your object in the Scene Hierarchy
• Click Export in the menu bar
• Choose OBJ format (universal compatibility)
• Pick a save location and filename
• Click Save
3 What Gets Exported
FormnexoStudio exports:
• .obj file (geometry, normals, UVs)
• .mtl file (material properties)
• PBR values (base color, metallic, roughness)
4 Import to Unity
• Drag the .obj file into Unity's Assets folder
• Unity will import the model and materials
• Assign textures manually if needed
• The model is ready to use in your game!
5 Import to Unreal Engine
• In Content Browser, click Import
• Select your .obj file
• Unreal will create a Static Mesh
• Create materials in Unreal using the PBR values from the .mtl file
Note: FBX and glTF export are coming soon! For now, OBJ works great for static meshes. For animated characters, export as FBX (currently exports as OBJ).
